About this Project

This project was completed by Professor Bill Barry's Advanced Greek class in fall 2017. Having discussed Herodotus' frequent digressions in class, we asked more questions: What do we consider to be a digression? How can we map these digressions? If we did map these digressions, what might the architecture of such a graph reveal?
